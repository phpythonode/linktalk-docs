# 虚拟声卡配置

虚拟声卡是软件之间的音频桥梁，遵循「Input 进声音 → Output 出声音」的配对规则。语桥采用**双链路隔离设计**，两个方向完全独立、互不干扰：

- **链路A（我→对方）**：真实麦克风 → 识别翻译 → 翻译输出（传给会议/直播）
- **链路B（对方→我）**：对方声音输入 → 识别翻译 → 对方翻译输出（本地听）

:::tip 核心原则
我的翻译走一个虚拟声卡，对方翻译走另一个虚拟声卡，**完全隔离，永不交叉**。
:::

---

## 需要几个虚拟声卡？

| 场景 | 所需数量 | 推荐方案 |
|------|----------|----------|
| 简单测试（单向） | 1个 | 只测「我→对方」或「对方→我」 |
| 单向直播/会议 | 1个 | 我的翻译声音传给观众/会议 |
| 双向国际会议 | 2个 | 两个方向完全隔离，无串音 |
| 复杂多流场景 | 3+个 | 按需给每个方向分配独立虚拟声卡 |

## 工具下载

### Windows：VB-Audio Cable（免费推荐）

下载 Cable A 和 Cable B 两个，安装后系统自动识别为两个独立虚拟声卡，轻量无广告。

- 官网：https://vb-audio.com/Cable/
- 可选：[VoiceMeeter Banana](https://vb-audio.com/Voicemeeter/)（可视化调试，方便排查无声/串音）

### macOS：BlackHole（免费推荐）

BlackHole 是 Mac 上最主流的虚拟声卡，原生支持 Apple 芯片（M1/M2/M3/M4）。

- 官网：https://existential.audio/blackhole/
- 推荐安装 **BlackHole 2ch**
- 也可通过 Homebrew 安装：
  ```bash
  brew install blackhole-2ch
  ```

:::tip Mac 双声卡方案
双向会议需要两个独立虚拟声卡时，安装 BlackHole 2ch + BlackHole 16ch 分别对应两条链路，或通过「音频 MIDI 设置」创建聚合设备。
:::

## 场景一：双向国际会议（完整互译）

**目标**：我说话 → 翻译后对方听到；对方说话 → 翻译后我听到，双向完全隔离。

**所需工具**：虚拟声卡A（我的翻译→会议）+ 虚拟声卡B（对方声音→同传）+ 真实麦克风 + 耳机

**链路A：我 → 对方**

```
真实麦克风 → 语桥「麦克风输入」→ 识别翻译
→ 语桥「翻译输出」→ 虚拟声卡A Input
→ 虚拟声卡A Output → 会议软件「麦克风输入」→ 对方听到翻译
```

**链路B：对方 → 我**

```
对方说话 → 会议软件「输出设备」→ 虚拟声卡B Input
→ 虚拟声卡B Output → 语桥「对方声音输入」→ 识别翻译
→ 语桥「对方翻译输出」→ 真实耳机 → 我听到翻译
```

**配置对照表**

| 软件 | 配置项 | 选择设备 |
|------|--------|----------|
| 语桥 | 麦克风输入 | 真实麦克风 |
| 语桥 | 翻译输出 | 虚拟声卡A Input |
| 语桥 | 我的翻译播放 | 关闭（避免回声） |
| 语桥 | 对方声音输入 | 虚拟声卡B Output |
| 语桥 | 对方翻译输出 | 真实耳机 |
| 会议软件 | 麦克风输入 | 虚拟声卡A Output |
| 会议软件 | 输出设备 | 虚拟声卡B Input |

## 场景二：单向直播/会议（我→对方）

适合演讲、直播带货，只需传出我的翻译，无需处理对方声音。

| 软件 | 配置项 | 选择设备 |
|------|--------|----------|
| 语桥 | 麦克风输入 | 真实麦克风 |
| 语桥 | 翻译输出 | 虚拟声卡A Input |
| 语桥 | 我的翻译播放 | 关闭 |
| OBS/会议软件 | 麦克风输入 | 虚拟声卡A Output |

## 常见误区

**❌ 误区1：两条链路用同一个虚拟声卡 → 死循环**

我的翻译输出和对方声音输入选了同一个虚拟声卡，导致翻译声音被反复采集，回声爆炸。  
✅ 解决：两条链路必须用**不同**的虚拟声卡。

**❌ 误区2：会议软件麦克风选真实麦克风 → 传原文**

会议里的人听到的是未翻译的原文，失去同传意义。  
✅ 解决：会议软件麦克风必须选**虚拟声卡A Output**。

**❌ 误区3：对方翻译输出选虚拟声卡 → 串音给会议**

对方翻译声音被会议软件采集传出去，造成串音。  
✅ 解决：对方翻译输出必须选**真实耳机**，只在本地播放。
