## Windows 安装

1. 下载 `LinkTalk_1.0.0_x64_en-US.msi`
2. 双击运行，按提示完成安装
3. 启动语桥，开始使用

---

## macOS 安装

1. 下载 `LinkTalk_1.0.0_aarch64.dmg`
2. 双击挂载，将 LinkTalk 拖入 Applications 文件夹
3. 由于应用未经 Apple 公证，首次需要执行以下命令解除限制：

```bash
xattr -cr /Applications/Linktalk.app
```

4. 打开终端，粘贴上方命令，回车执行
5. 执行完成后即可正常启动语桥

:::tip
如果提示"无法打开，因为无法验证开发者"，执行上方命令后重新打开即可。
:::
