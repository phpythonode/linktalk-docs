# 国际会议

中英双向实时互译，接入 Zoom / 腾讯会议 / Teams，两端参会者各听各的母语。

---

## 适用情况

- 参会者中英文混杂，需要双向翻译
- 不想为一场会议支付数千元的人工同传费用
- 使用 Zoom、腾讯会议、飞书等主流会议软件

---

## 工作方式

语桥双通道架构天然适配双向会议场景：

- **通道 A**：我说中文 → 英文输出 → 传入会议，对方听英文
- **通道 B**：对方说英文 → 中文输出 → 传入耳机，我听中文

两条链路完全隔离，不会互相干扰，也不会产生回声或死循环。

支持中英混合识别（zhen 模式），「大家好，今天聊 AI 趋势」这类中英夹杂的表达也能准确识别和翻译。
