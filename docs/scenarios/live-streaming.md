# 跨境直播

用中文完成多语言直播，无需外语主播，观众实时听到母语。

---

## 适用情况

- 想开拓东南亚、欧美、日本等海外直播市场
- 不想依赖外语主播（成本高、稳定性差）
- 希望用自己的声音和风格与海外观众互动

---

## 工作方式

你正常用中文说话 → 语桥实时识别 → 翻译为目标语言 → 以你的音色输出 → 通过虚拟声卡传入直播推流。

**首音延迟 1.3 秒**，观众几乎感知不到停顿。音色克隆无需提前训练，开播即生效，保留你的语气和情绪。

---

## 与雇佣外语主播的对比

| 维度 | 外语主播 | 语桥 |
|------|----------|------|
| 时薪成本 | 200–500 元 | 4–5.5 元 |
| 语言覆盖 | 1 种 | 10+ 种 |
| 稳定性 | 易请假/跳槽 | 7×24 小时 |
| 声音风格 | 他人风格 | 你自己的音色 |
| 启动门槛 | 招聘+培训 | 下载即用 |

---

## 推荐配置

1. 麦克风 → 语桥输入
2. 语桥翻译输出 → 虚拟声卡 A Input
3. OBS / 直播伴侣麦克风 → 虚拟声卡 A Output
4. 开播，用中文正常说话

详细音频路由参考[虚拟声卡配置](../virtual-audio)。
