常见问题

延迟有多少？​

实测首音延迟约 1.3 秒。市面主流同传工具普遍在 3 秒左右，语桥快约 2 倍。

这个延迟来自火山引擎 API 的识别+翻译时间加上语桥的流式播放优化。对直播和会议场景来说，1.3 秒基本感知不到停顿。

支持 Mac 吗？​

支持。语桥提供 macOS Apple 芯片原生版，M1/M2/M3/M4 均可运行，无需 Rosetta 转译。Intel Mac 暂不支持。

双通道怎么理解？​

可以把它理解为两条完全独立的翻译管道，各自有独立的输入、输出和声卡配置，互不干扰。

典型用法：

  • 通道 A 负责「我说 → 对方听」
  • 通道 B 负责「对方说 → 我听」

两条链路同时运行，实现真正的双向互译。

费用怎么计算？​

语桥本身不按时间或功能收费，费用来自火山引擎 API 的 token 消耗：

  • 按实际语音时长计费
  • 新用户赠送 100 万 token，约 29 小时，完全够前期使用
  • 不使用不产生费用

需要显卡吗？​

不需要。语桥的识别和翻译全部在云端完成，本地只负责音频采集和播放，普通集显笔记本即可流畅运行。

音色克隆需要提前训练吗？​

不需要。语桥采用实时克隆方案，开口即生效，无需录制样本或等待训练。如需深度定制专属音色，请联系客服。

支持哪些语言？​

目前支持：中文、英语、日语、印尼语、西班牙语、葡萄牙语、德语、法语，以及中英混合模式（zhen）。

更多语言持续开发中，有需求可在社群反馈，优先排期。

能对接本地离线版吗？​

可以接入本地语音识别和 TTS，但效果与云端方案差距较大：本地方案对硬件要求高、识别准确率低、音色克隆效果差。语桥云端方案在延迟和质量上均优于目前市面上的本地方案。

杀毒软件报警怎么办？​

属于误报，建议将语桥加入杀毒软件的信任区（推荐使用火绒）。
