# Cas d'Usage

LinkTalk est optimisé pour trois scénarios à haute fréquence entre langues.

| Scénario | Besoin Principal | Configuration Recommandée |
|----------|----------------|--------------------------|
| [Export de Contenu](./content-export) | Vidéo en mandarin → versions doublées en plusieurs langues | Sortie canal unique |
| [Streaming Transfrontalier](./live-streaming) | Parler mandarin, l'audience entend sa langue maternelle | Canal unique + clonage vocal |
| [Réunions Internationales](./conference) | Interprétation bidirectionnelle Mandarin-Anglais en temps réel | Double canal + audio virtuel |
