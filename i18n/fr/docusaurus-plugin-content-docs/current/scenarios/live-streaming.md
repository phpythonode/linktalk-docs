# Streaming Transfrontalier

Diffusez en mandarin pour un public mondial dans sa langue maternelle — sans hôte en langue étrangère.

---

## Comment ça Fonctionne

Vous parlez mandarin → LinkTalk reconnaît en temps réel → traduit dans la langue cible → sortie avec votre voix clonée → routé via un périphérique audio virtuel vers le logiciel de streaming.

**Latence initiale : 1,3 seconde** — le public remarque à peine une pause.

---

## Configuration Recommandée

1. Microphone → entrée LinkTalk
2. Sortie traduction LinkTalk → Virtual Device A Input
3. Microphone OBS / logiciel de streaming → Virtual Device A Output
4. Commencer à diffuser et parler mandarin normalement

Voir [Configuration Audio Virtuelle](../virtual-audio) pour les détails de routage audio.
