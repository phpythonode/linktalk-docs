# Réunions Internationales

Interprétation bidirectionnelle Mandarin-Anglais en temps réel, intégrée avec Zoom / Tencent Meeting / Teams.

---

## Comment ça Fonctionne

- **Canal A** : Je parle mandarin → sortie en anglais → envoyé à la réunion → ils entendent l'anglais
- **Canal B** : Ils parlent anglais → sortie en mandarin → envoyé au casque → j'entends le mandarin
