## ステップ 3：API の設定

語橋は Volcengine 同時通訳 API をベースエンジンとして使用します。

1. アカウント登録：https://console.volcengine.com/auth/login
2. コンソールに入り、**同時通訳**サービスを有効化
3. API キーを作成し、コピーして語橋の設定ページに貼り付け

:::tip 新規ユーザー無料クレジット
Volcengine は新規ユーザーに **100 万トークン（約 29 時間）** を無料で付与します。日常使用には十分です。
:::

:::caution
API キーを設定する際は、**VPN やプロキシを無効にしてください**。そうしないと接続に失敗する可能性があります。
:::

---

## ステップ 4：翻訳開始

1. 設定で入力言語と出力言語を選択
2. 音声入力デバイス（マイク）と出力デバイスを設定
3. 開始をクリックし、マイクに向かって話すと翻訳がリアルタイムで出力されます

翻訳音声をライブ配信や会議ソフトに送る方法は[仮想オーディオ設定](./virtual-audio)を参照してください。

## システム要件

| 項目 | 要件 |
|------|------|
| Windows | Windows 10 / 11（64 ビット） |
| macOS | Apple Silicon（M1 / M2 / M3 / M4） |
| オーディオ | マイク + スピーカーまたはヘッドフォン |
| ネットワーク | 遅延 < 100ms 推奨、有線または安定した Wi-Fi |
| GPU | 不要、通常の内蔵グラフィックスで可 |
