# よくある質問

## 遅延はどのくらいですか？

実測初音遅延は約 1.3 秒です。市場の主流同時通訳ツールは通常 3 秒前後で、語橋は約 2 倍高速です。

## Mac に対応していますか？

はい。語橋は macOS Apple Silicon ネイティブ版を提供しており、M1/M2/M3/M4 で Rosetta なしで動作します。Intel Mac は現在未対応です。

## デュアルチャンネルとは何ですか？

2 つの完全独立した翻訳パイプラインと考えてください。各自独立した入力、出力、音声デバイス設定を持ち、互いに干渉しません。

典型的な使い方：
- チャンネル A：「私が話す → 相手が聞く」
- チャンネル B：「相手が話す → 私が聞く」

両方が同時に動作し、真の双方向翻訳を実現します。

## 料金はどのように計算されますか？

語橋自体は時間や機能による課金はありません。費用は Volcengine API のトークン消費から発生します：

- 実際の音声時間に基づいて課金
- 新規ユーザーには 100 万トークン（約 29 時間）が無料で付与
- 使用しなければ費用は発生しません

## GPU は必要ですか？

不要です。語橋の認識と翻訳はすべてクラウドで処理されます。ローカルデバイスは音声の収集と再生のみを担当するため、通常の内蔵グラフィックスのノートパソコンでもスムーズに動作します。

## 音声クローンに事前トレーニングは必要ですか？

不要です。語橋はリアルタイムクローン方式を採用しており、話し始めた瞬間から有効になります。サンプル録音やトレーニング待ちは不要です。

## 対応言語は何ですか？

現在対応：中国語、英語、日本語、インドネシア語、スペイン語、ポルトガル語、ドイツ語、フランス語、および中英混合モード（zhen）。

## ウイルス対策ソフトが警告を出した場合は？

誤検知です。語橋をウイルス対策ソフトの信頼リストに追加してください。
