# 越境ライブ配信

中国語で多言語ライブ配信、外国語ホスト不要、視聴者はリアルタイムで母国語を聞けます。

---

## 適用状況

- 東南アジア、欧米、日本などの海外ライブ配信市場を開拓したい
- 外国語ホストに依存したくない（コストが高く、安定性が低い）
- 自分の声とスタイルで海外視聴者と交流したい

---

## 動作方式

通常通り中国語で話す → 語橋がリアルタイムで認識 → ターゲット言語に翻訳 → あなたの音声クローンで出力 → 仮想オーディオデバイスを通じてライブ配信ソフトに送信。

**初音遅延 1.3 秒**、視聴者はほとんど停止を感じません。音声クローンは事前トレーニング不要、配信開始と同時に有効になり、あなたのトーンと感情を保持します。

---

## 外国語ホスト雇用との比較

| 次元 | 外国語ホスト | 語橋 |
|------|------------|------|
| 時給コスト | 200〜500 元/時間 | 月額 ¥88 から |
| 言語カバレッジ | 1 言語 | 10 以上の言語 |
| 安定性 | 休暇/退職しやすい | 7×24 時間 |
| 声のスタイル | 他人の声 | あなた自身の声 |
| 開始ハードル | 採用+トレーニング | ダウンロードしてすぐ使える |

---

## 推奨設定

1. マイク → 語橋入力
2. 語橋翻訳出力 → 仮想デバイス A Input
3. OBS / 配信ソフトのマイク → 仮想デバイス A Output
4. 配信開始、通常通り中国語で話す

詳細な音声ルーティングは[仮想オーディオ設定](../virtual-audio)を参照してください。
