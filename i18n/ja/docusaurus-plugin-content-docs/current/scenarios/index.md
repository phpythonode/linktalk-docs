# ユースケース

語橋は 3 つの高頻度クロスランゲージシナリオに特化して最適化されています。ニーズに合ったセクションを選択してください。各シナリオには完全な設定ガイドとコスト参考が含まれています。

| シナリオ | コアニーズ | 推奨設定 |
|----------|-----------|---------|
| [コンテンツ輸出](./content-export) | 中国語動画 → 多言語吹き替え版 | シングルチャンネル出力 |
| [越境ライブ配信](./live-streaming) | 中国語で話す、視聴者は母国語で聞く | シングルチャンネル + 音声クローン |
| [国際会議](./conference) | 中英双方向リアルタイム通訳 | デュアルチャンネル + 仮想オーディオ |
