# 技術仕様

## 遅延パフォーマンス

| 指標 | 数値 |
|------|------|
| 初音遅延 | **1.3 秒**（実測） |
| 業界平均 | 約 3 秒 |
| 優位性 | 2 倍以上高速 |

技術基盤：Volcengine 同時通訳 API + リアルタイムストリーミング最適化。

## システム要件

| 項目 | 要件 |
|------|------|
| OS | Windows 10/11（64 ビット）/ macOS（Apple Silicon） |
| オーディオ | マイク + スピーカー/ヘッドフォン |
| ネットワーク | 遅延 < 100ms、有線または安定した Wi-Fi 推奨 |
| GPU | **不要** |
