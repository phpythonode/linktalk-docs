# LinkTalk · Interpretasi Simultan Real-Time

LinkTalk adalah **alat interpretasi simultan real-time desktop** untuk kreator konten dan bisnis lintas batas, mendukung Windows dan macOS (Apple Silicon).

Kemampuan inti: **Terjemahan dual-channel independen · Kloning suara real-time · Latensi suara pertama 1,3 detik**

---

## Mengapa LinkTalk?

### Lintas platform: Windows dan Mac keduanya didukung

Sebagian besar alat interpretasi di pasaran hanya mendukung Windows. LinkTalk menyediakan versi Windows dan versi native macOS Apple Silicon — M1/M2/M3/M4 semuanya berjalan lancar tanpa konfigurasi tambahan.

### Terjemahan dua arah sejati dengan channel yang sepenuhnya terisolasi

LinkTalk menggunakan arsitektur dual-channel terisolasi. Channel A dan Channel B berjalan secara independen tanpa saling mengganggu. Mendukung tidak hanya output satu arah ("saya bicara → mereka dengar") tetapi juga terjemahan dua arah penuh.

### Latensi suara pertama 1,3 detik — terdepan di industri

Dibangun di atas Volcengine API dengan optimasi streaming mendalam, latensi suara pertama yang terukur adalah 1,3 detik, sementara alat mainstream biasanya sekitar 3 detik.

### Kloning suara real-time — terdengar seperti Anda

Tidak perlu merekam sampel atau melatih model terlebih dahulu — kloning dimulai saat Anda mulai berbicara. Output terjemahan mempertahankan nada, emosi, dan ritme Anda.

### Bayar sesuai penggunaan dengan kredit gratis untuk pengguna baru

Didukung oleh Volcengine API resmi. Pengguna baru mendapatkan 1.000.000 token gratis (sekitar 29 jam).

## Ikhtisar Fitur

| Fitur | Deskripsi |
|-------|-----------|
| Terjemahan dual-channel | Dua channel sepenuhnya independen, masing-masing terikat ke perangkat audio sendiri |
| Kloning suara real-time | Tidak perlu pelatihan — kloning dimulai segera, mempertahankan emosi dan ritme |
| Latensi 1,3 detik | Optimasi streaming, 2× lebih cepat dari rata-rata industri |
| Subtitle real-time bilingual | Terintegrasi dengan OBS / perangkat lunak streaming |
| Pengenalan campuran Mandarin-Inggris | Mode `zhen` mengenali input campuran secara otomatis |
| Dukungan multi-bahasa | Mandarin, Inggris, Jepang, Indonesia, Spanyol, Portugis, Jerman, Prancis |
| Lintas platform | Windows 10/11 · macOS Apple Silicon (M1–M4) |
