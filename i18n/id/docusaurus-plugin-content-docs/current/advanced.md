# Penggunaan Lanjutan

---

## Integrasi dengan OBS / Perangkat Lunak Streaming

Merutekan output terjemahan LinkTalk ke OBS memungkinkan:

- Subtitle bilingual real-time di atas video Anda
- Terjemahan simultan selama streaming game tanpa berpindah jendela
- Perekaman audio multi-track untuk pasca-produksi yang fleksibel

---

## Konfigurasi Pasangan Bahasa Dual-Channel Independen

Dua channel dapat dikonfigurasi dengan pasangan bahasa yang berbeda, misalnya:

- Channel A: Mandarin → Inggris (untuk audiens Barat)
- Channel B: Mandarin → Jepang (untuk audiens Jepang)

Keduanya berjalan bersamaan, masing-masing terikat ke perangkat output audio independen.

---

## Pengenalan Campuran Mandarin-Inggris (Mode zhen)

Pilih `zhen` di pengaturan bahasa dan LinkTalk akan secara otomatis mengenali input campuran Mandarin-Inggris tanpa perlu beralih bahasa secara manual.

---

## Pergantian Suara

LinkTalk mendukung pergantian preset suara selama terjemahan tanpa perlu restart.
