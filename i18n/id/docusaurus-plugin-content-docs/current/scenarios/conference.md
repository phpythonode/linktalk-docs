# Rapat Internasional

Interpretasi dua arah Mandarin-Inggris real-time, terintegrasi dengan Zoom / Tencent Meeting / Teams.

---

## Cara Kerja

- **Channel A**: Saya bicara Mandarin → output Inggris → dikirim ke rapat → mereka dengar Inggris
- **Channel B**: Mereka bicara Inggris → output Mandarin → dikirim ke headphone → saya dengar Mandarin
