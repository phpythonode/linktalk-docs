# Streaming Langsung Lintas Batas

Streaming dalam bahasa Mandarin ke audiens global dalam bahasa ibu mereka — tanpa host berbahasa asing.

---

## Cara Kerja

Anda bicara Mandarin → LinkTalk mengenali secara real-time → menerjemahkan ke bahasa target → output dengan suara kloning Anda → dikirim melalui perangkat audio virtual ke perangkat lunak streaming.

**Latensi suara pertama: 1,3 detik** — audiens hampir tidak merasakan jeda.

---

## Pengaturan yang Disarankan

1. Mikrofon → input LinkTalk
2. Output terjemahan LinkTalk → Virtual Device A Input
3. Mikrofon OBS / perangkat lunak streaming → Virtual Device A Output
4. Mulai siaran dan bicara Mandarin seperti biasa

Lihat [Pengaturan Audio Virtual](../virtual-audio) untuk detail routing audio.
