# Kasus Penggunaan

LinkTalk dioptimalkan khusus untuk tiga skenario lintas bahasa frekuensi tinggi.

| Skenario | Kebutuhan Inti | Pengaturan yang Disarankan |
|----------|---------------|--------------------------|
| [Ekspor Konten](./content-export) | Video Mandarin → versi dubbing multi-bahasa | Output single-channel |
| [Streaming Lintas Batas](./live-streaming) | Bicara Mandarin, audiens dengar bahasa ibu | Single channel + kloning suara |
| [Rapat Internasional](./conference) | Interpretasi dua arah Mandarin-Inggris real-time | Dual-channel + audio virtual |
