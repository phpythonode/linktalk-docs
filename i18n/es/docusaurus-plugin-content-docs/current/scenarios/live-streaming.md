# Streaming Transfronterizo

Transmite en mandarín a una audiencia global en su idioma nativo — sin host en idioma extranjero.

---

## Cómo Funciona

Hablas mandarín → LinkTalk reconoce en tiempo real → traduce al idioma objetivo → salida con tu voz clonada → enrutado a través de un dispositivo de audio virtual al software de streaming.

**Latencia inicial: 1,3 segundos** — la audiencia apenas nota ninguna pausa.

---

## Configuración Recomendada

1. Micrófono → entrada de LinkTalk
2. Salida de traducción de LinkTalk → Virtual Device A Input
3. Micrófono de OBS / software de streaming → Virtual Device A Output
4. Comienza a transmitir y habla mandarín normalmente

Ver [Configuración de Audio Virtual](../virtual-audio) para detalles de enrutamiento de audio.
