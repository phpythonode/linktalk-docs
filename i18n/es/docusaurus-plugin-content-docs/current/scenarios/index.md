# Casos de Uso

LinkTalk está optimizado para tres escenarios de alta frecuencia entre idiomas.

| Escenario | Necesidad Principal | Configuración Recomendada |
|-----------|--------------------|-----------------------------|
| [Exportación de Contenido](./content-export) | Video en mandarín → versiones dobladas en múltiples idiomas | Salida de canal único |
| [Streaming Transfronterizo](./live-streaming) | Habla mandarín, audiencia escucha su idioma nativo | Canal único + clonación de voz |
| [Reuniones Internacionales](./conference) | Interpretación bidireccional Mandarín-Inglés en tiempo real | Doble canal + audio virtual |
