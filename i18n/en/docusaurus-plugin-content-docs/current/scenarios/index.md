# Use Cases

LinkTalk is purpose-built for three high-frequency cross-language scenarios. Choose the section that matches your needs — each includes a complete setup guide and cost reference.

| Scenario | Core Need | Recommended Setup |
|----------|-----------|-------------------|
| [Content Export](./content-export) | Chinese video → multi-language dubbed versions | Single-channel output |
| [Cross-Border Live Streaming](./live-streaming) | Speak Chinese, audience hears their native language | Single channel + voice cloning |
| [International Meetings](./conference) | Real-time bidirectional Chinese-English interpretation | Dual-channel + virtual audio |
