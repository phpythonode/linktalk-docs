# Technical Specifications

## Latency

| Metric | Value |
|--------|-------|
| First-audio latency | **1.3 seconds** (measured) |
| Industry average | ~3 seconds |
| Advantage | 2× faster |

Powered by the Volcengine simultaneous interpretation API with real-time streaming optimization.
