# LinkTalk · Echtzeit-Simultandolmetschung

LinkTalk ist ein **Desktop-Echtzeit-Simultandolmetsch-Tool** für Content-Creator und grenzüberschreitende Unternehmen, kompatibel mit Windows und macOS (Apple Silicon).

Kernfähigkeiten: **Unabhängige Dual-Channel-Übersetzung · Echtzeit-Stimmklonen · Erstton-Latenz 1,3 Sekunden**

---

## Warum LinkTalk?

### Plattformübergreifend: Windows und Mac beide unterstützt

Die meisten Dolmetsch-Tools unterstützen nur Windows. LinkTalk bietet sowohl eine Windows-Version als auch eine native macOS Apple Silicon-Version — M1/M2/M3/M4 laufen alle ohne zusätzliche Konfiguration reibungslos.

### Echte bidirektionale Übersetzung mit vollständig isolierten Kanälen

LinkTalk verwendet eine Dual-Channel-Isolierungsarchitektur. Kanal A und Kanal B laufen unabhängig voneinander ohne gegenseitige Störung.

### Erstton-Latenz 1,3 Sekunden — branchenführend

Basierend auf der Volcengine API mit tiefer Streaming-Optimierung beträgt die gemessene Erstton-Latenz 1,3 Sekunden, während gängige Tools typischerweise bei etwa 3 Sekunden liegen.

### Echtzeit-Stimmklonen — klingt wie Sie

Kein Aufnehmen von Samples oder Trainieren von Modellen im Voraus — das Klonen beginnt in dem Moment, in dem Sie zu sprechen beginnen.

### Nutzungsbasierte Abrechnung mit kostenlosen Credits für neue Nutzer

Betrieben von der offiziellen Volcengine API. Neue Nutzer erhalten 1.000.000 kostenlose Token (ca. 29 Stunden).

## Funktionsübersicht

| Funktion | Beschreibung |
|---------|--------------|
| Dual-Channel-Übersetzung | Zwei vollständig unabhängige Kanäle |
| Echtzeit-Stimmklonen | Kein Training — sofortiges Klonen |
| 1,3 s Latenz | Streaming-Optimierung, 2× schneller als Durchschnitt |
| Zweisprachige Echtzeit-Untertitel | Integration mit OBS / Streaming-Software |
| Gemischte Mandarin-Englisch-Erkennung | `zhen`-Modus erkennt gemischte Eingaben automatisch |
| Mehrsprachige Unterstützung | Mandarin, Englisch, Japanisch, Indonesisch, Spanisch, Portugiesisch, Deutsch, Französisch |
| Plattformübergreifend | Windows 10/11 · macOS Apple Silicon (M1–M4) |

## Anwendungsfälle

- **Content-Export**: Mehrsprachige Synchronversionen von Mandarin-Videos erstellen
- **Grenzüberschreitendes Streaming**: In Mandarin für ein globales Publikum streamen
- **Internationale Meetings**: Echtzeit-Bidirektionale Mandarin-Englisch-Dolmetschung
