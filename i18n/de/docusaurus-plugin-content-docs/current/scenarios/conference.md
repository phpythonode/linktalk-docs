# Internationale Meetings

Echtzeit-Bidirektionale Mandarin-Englisch-Dolmetschung, integriert mit Zoom / Tencent Meeting / Teams.

---

## Wie es funktioniert

- **Kanal A**: Ich spreche Mandarin → Englische Ausgabe → an Meeting gesendet → sie hören Englisch
- **Kanal B**: Sie sprechen Englisch → Mandarin-Ausgabe → an Kopfhörer gesendet → ich höre Mandarin
