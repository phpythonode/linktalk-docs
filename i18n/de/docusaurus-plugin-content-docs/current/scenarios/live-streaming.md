# Grenzüberschreitendes Streaming

Streamen Sie auf Mandarin für ein globales Publikum in seiner Muttersprache — ohne fremdsprachigen Host.

---

## Wie es funktioniert

Sie sprechen Mandarin → LinkTalk erkennt in Echtzeit → übersetzt in die Zielsprache → Ausgabe mit Ihrer geklonten Stimme → über virtuelles Audiogerät an Streaming-Software weitergeleitet.

**Erstton-Latenz: 1,3 Sekunden** — das Publikum bemerkt kaum eine Pause.

---

## Empfohlene Einrichtung

1. Mikrofon → LinkTalk-Eingang
2. LinkTalk-Übersetzungsausgabe → Virtual Device A Input
3. OBS / Streaming-Software-Mikrofon → Virtual Device A Output
4. Streaming starten und normal Mandarin sprechen

Siehe [Virtuelle Audio-Einrichtung](../virtual-audio) für Audio-Routing-Details.
