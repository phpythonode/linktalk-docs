# Anwendungsfälle

LinkTalk ist für drei hochfrequente sprachübergreifende Szenarien optimiert.

| Szenario | Kernbedarf | Empfohlene Einrichtung |
|----------|-----------|------------------------|
| [Content-Export](./content-export) | Mandarin-Video → mehrsprachige Synchronversionen | Einzelkanal-Ausgabe |
| [Grenzüberschreitendes Streaming](./live-streaming) | Mandarin sprechen, Publikum hört Muttersprache | Einzelkanal + Stimmklonen |
| [Internationale Meetings](./conference) | Echtzeit-Bidirektionale Mandarin-Englisch-Dolmetschung | Dual-Channel + virtuelles Audio |
