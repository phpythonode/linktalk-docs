# Häufige Fragen

## Wie hoch ist die Latenz?

Die gemessene Erstton-Latenz beträgt ca. 1,3 Sekunden. Gängige Tools liegen typischerweise bei 3 Sekunden — LinkTalk ist ca. 2× schneller.

## Wird Mac unterstützt?

Ja. LinkTalk bietet eine native macOS Apple Silicon-Version, die auf M1/M2/M3/M4 ohne Rosetta läuft. Intel-Macs werden derzeit nicht unterstützt.

## Wie funktioniert Dual-Channel?

Stellen Sie es sich als zwei vollständig unabhängige Übersetzungs-Pipelines vor, jede mit eigener Eingabe-, Ausgabe- und Audiogerätekonfiguration.

## Wie wird der Preis berechnet?

LinkTalk selbst berechnet keine Zeit- oder Funktionsgebühren. Kosten entstehen durch den Token-Verbrauch der Volcengine API. Neue Nutzer erhalten 1.000.000 kostenlose Token (~29 Stunden).

## Wird eine GPU benötigt?

Nein. Erkennung und Übersetzung von LinkTalk werden alle in der Cloud verarbeitet.

## Erfordert Stimmklonen Vortraining?

Nein. LinkTalk verwendet einen Echtzeit-Klonansatz — wirksam in dem Moment, in dem Sie zu sprechen beginnen.

## Welche Sprachen werden unterstützt?

Derzeit: Mandarin, Englisch, Japanisch, Indonesisch, Spanisch, Portugiesisch, Deutsch, Französisch und gemischter Mandarin-Englisch-Modus (zhen).
