# Especificações Técnicas

## Latência

| Métrica | Valor |
|---------|-------|
| Latência inicial | **1,3 segundos** (medido) |
| Média do setor | ~3 segundos |
| Vantagem | 2× mais rápido |

## Requisitos do Sistema

| Item | Requisito |
|------|-----------|
| SO | Windows 10/11 (64 bits) / macOS (Apple Silicon) |
| Áudio | Microfone + alto-falantes/fones de ouvido |
| Rede | Latência < 100ms, cabo ou Wi-Fi estável recomendado |
| GPU | **Não necessária** |
