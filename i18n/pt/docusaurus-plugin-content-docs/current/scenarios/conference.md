# Reuniões Internacionais

Interpretação bidirecional Mandarim-Inglês em tempo real, integrada com Zoom / Tencent Meeting / Teams.

---

## Como Funciona

- **Canal A**: Falo mandarim → saída em inglês → enviado para a reunião → eles ouvem inglês
- **Canal B**: Eles falam inglês → saída em mandarim → enviado para fones de ouvido → eu ouço mandarim
