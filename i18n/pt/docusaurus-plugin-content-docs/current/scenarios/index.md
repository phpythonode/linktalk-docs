# Casos de Uso

LinkTalk é otimizado para três cenários de alta frequência entre idiomas.

| Cenário | Necessidade Principal | Configuração Recomendada |
|---------|----------------------|--------------------------|
| [Exportação de Conteúdo](./content-export) | Vídeo em mandarim → versões dubladas em múltiplos idiomas | Saída de canal único |
| [Streaming Transfronteiriço](./live-streaming) | Fale mandarim, audiência ouve seu idioma nativo | Canal único + clonagem de voz |
| [Reuniões Internacionais](./conference) | Interpretação bidirecional Mandarim-Inglês em tempo real | Canal duplo + áudio virtual |
