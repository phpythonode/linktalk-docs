# Streaming Transfronteiriço

Transmita em mandarim para uma audiência global no idioma nativo deles — sem host em idioma estrangeiro.

---

## Como Funciona

Você fala mandarim → LinkTalk reconhece em tempo real → traduz para o idioma alvo → saída com sua voz clonada → roteado através de um dispositivo de áudio virtual para o software de streaming.

**Latência inicial: 1,3 segundos** — a audiência mal percebe qualquer pausa.

---

## Configuração Recomendada

1. Microfone → entrada do LinkTalk
2. Saída de tradução do LinkTalk → Virtual Device A Input
3. Microfone do OBS / software de streaming → Virtual Device A Output
4. Comece a transmitir e fale mandarim normalmente

Ver [Configuração de Áudio Virtual](../virtual-audio) para detalhes de roteamento de áudio.
